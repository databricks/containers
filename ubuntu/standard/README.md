# Standard Container

This image is intended to have feature parity with the current Databricks Runtime.
Support for features will be added over time.

## Supported Features
  - Scala Notebooks
  - Java/Jar jobs
  - Python Notebooks, Python Jobs
  - Spark Submit Jobs
  - %sh
  - DBFS FUSE mount (/dbfs)

## Unsupported Features
  - SSH
  - Ganglia
